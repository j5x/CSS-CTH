Hallo wereld

Eerst moet je een github repository aanmaken om github waar jij je bestand wilt uploaden.

Bijv https://github.com/j5x/test.

Dan open je CMD of Terminal en dan ga je jezelf locaten in de map waar je wilt zijn.

Je gebruikt cd.. om terug te gaan naar de vorige map en cd (naammap) om naar die map toe te gaan Als je nog geen map hebt:

- Typ in windows search bar: CMD

- In de terminal typ je mkdir dan maak je een folder aan in die directory

- Dan typ je explorer . om te kijken je hem kan openen

Als je bij de folder hebt aangemaakt:

Typ in explorer navigation bar cmd om de directory te openen in cmd/terminal.

- Andere optie: type cmd in windows search

- Ga naar de map door cd naammap te gebruiken en cd.. om terug te gaan

- git init dit maakt een .git folder aan in de map.

- git add (naambestand)

- git commit -m “first commit”

- git branch -M main

- git remote add origin https://github.com/j5x/test.git

- git push -u origin main
